I am a Network Security Engineer focused on securing and troubleshooting enterprise networks across Microsoft Azure, data centers, campuses, and remote locations.

My primary specialization is **Palo Alto network security**, **Azure cloud networking**, and **cloud security architecture**. I focus on traffic-flow analysis, firewall policy design, hybrid-cloud connectivity, routing, segmentation, logging, validation, and clear technical documentation.

## Engineering Principles

- Understand the complete traffic flow before changing policy or routing
- Prefer least privilege without creating unnecessary operational risk
- Validate both successful and denied traffic paths
- Build rollback and recovery into every change
- Document decisions, assumptions, and tradeoffs
- Never publish employer, customer, credential, or production-environment information
